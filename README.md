# DNA 유전자 검사 기반 보험설계사 모집 랜딩페이지

## 📋 프로젝트 개요

DNA 유전자 검사를 통한 DB 제공, 전문 교육 및 자격증 발급, 온라인 마케팅 교육을 제공하는 보험설계사 모집 랜딩페이지입니다.

### 주요 특징
- ✨ 현대적이고 반응형 디자인
- 🎯 월 500만원 이상, 연봉 1억 이상의 수익 목표 강조
- 📝 신청자 정보 수집 및 관리 시스템
- 📊 관리자 페이지를 통한 신청자 DB 관리
- 💾 LocalStorage 기반 데이터 저장

## 🚀 주요 기능

### 메인 랜딩페이지 (index.html)
1. **Hero Section**: 강력한 메시지와 CTA 버튼
2. **수익 하이라이트**: 월 500만원+, 연봉 1억+ 강조
3. **혜택 섹션**: 6가지 핵심 혜택 소개
4. **프로세스**: 4단계 시작 프로세스
5. **타겟 고객**: 이런 분들께 추천
6. **성공 스토리**: 실제 사례 (테스티모니얼)
7. **신청 폼**: 상세한 정보 수집 양식
8. **Final CTA**: 추가 전환 유도

### 관리자 페이지 (admin.html)
1. **통계 대시보드**: 총 신청자, 대기중, 연락완료, 합격 통계
2. **검색 기능**: 이름, 연락처, 이메일로 검색
3. **필터링**: 상태별, 수익목표별 필터
4. **정렬**: 최신순, 오래된순, 이름순
5. **신청자 상세보기**: 모든 정보 확인
6. **상태 관리**: 대기중/연락완료/합격/불합격 상태 변경
7. **CSV 내보내기**: 데이터 엑셀 다운로드
8. **삭제 기능**: 개별/전체 삭제


## 🔧 사용 방법

### 1. 웹사이트 실행

#### Python 서버 사용:
```bash
cd /home/user/webapp
python3 -m http.server 8000
```

#### Node.js 서버 사용:
```bash
cd /home/user/webapp
npx http-server -p 8000
```

### 2. 페이지 접속

- **메인 랜딩페이지**: http://localhost:8000/index.html
- **관리자 페이지**: http://localhost:8000/admin.html

### 3. 신청자 정보 입력

메인 페이지의 신청 폼에서 다음 정보를 입력:
- 이름, 연락처, 이메일 (필수)
- 나이, 성별, 주소 (필수)
- 보험 경력 (선택)
- 지원 동기 (필수)
- 수익 목표 (필수)
- 관심있는 마케팅 채널 (선택)

### 4. 관리자 페이지에서 DB 관리

- 신청자 목록 조회
- 검색 및 필터링
- 상세 정보 확인
- 상태 변경 (대기중 → 연락완료 → 합격/불합격)
- CSV로 데이터 내보내기
